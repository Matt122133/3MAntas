# Analysis

Esta carpeta guarda los registros de trabajo de forma persistente y estructurada.

## Qué guardar aquí

- Pasos realizados durante una tarea.
- Hallazgos importantes o decisiones temporales.
- Evidencia de verificación y resultados.
- Notas de seguimiento para continuar más tarde.

## Reglas de uso

- Usar un archivo por tarea o hilo de trabajo cuando el contexto lo amerite.
- Mantener el contenido claro, corto y accionable.
- Preferir hechos verificados sobre suposiciones.
- Si el contenido se vuelve relevante para el proyecto, moverlo a un ADR, plan o documentación permanente.

## Plantilla rápida

Copiar esta estructura al crear un nuevo análisis:

```md
# Analysis: <tema>

## Objetivo

## Pasos realizados

1. ...
2. ...

## Hallazgos

## Verificación

## Siguientes pasos
```
